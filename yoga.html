Yoga Quiz

DEMO | CODE

Project:

Namaste, friends! This quiz takes a user through five questions about Yoga. If the user quesses incorrectly, the correct answer is shown. The quiz also keeps track of how many quesitons the user guessed correctly. Yoga is one of my favorite activies, so this project was a blast to work on. My biggest challenge transitioning from hard-coding the quiz to dynamically programming the questions with object-oriented JavaSrcipt. Moving forward, I would improve on this project by adding more UX elements such as smoother transitions and animation effects.
